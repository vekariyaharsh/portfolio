# Portfolio

Welcome to **Vekariyaharsh's Portfolio** repository.

## Overview
This project showcases my personal portfolio website built with vanilla HTML, CSS, and JavaScript. It includes:
- A modern, responsive design with vibrant colors and smooth micro‑animations.
- Sections for **About**, **Projects**, **Expertise**, and **Contact**.
- Integration with **Firebase Hosting** for easy deployment.

## Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/vekariyaharsh/portfolio.git
   cd portfolio
   ```
2. **Install dependencies** (if any) – this project uses only static assets, so no additional packages are required.
3. **Run locally**
   ```bash
   # Open index.html in a browser or use a simple server
   npx -y serve .
   ```
4. **Deploy to Firebase** (see the Firebase setup guide in the repository).
